1) Получить платеж
curl -X GET 'http://localhost:8080/api/payments/123e4567-e89b-12d3-a456-426614174000'

2) Создать платеж
curl -X POST 'http://localhost:8080/api/payments' \
  -H 'Content-Type: application/json' \
  -d '{
    "inquiryRefId": "123e4567-e89b-12d3-a456-426614174001",
    "amount": 200.00,
    "currency": "EUR",
    "transactionRefId": "123e4567-e89b-12d3-a456-426614174003",
    "status": "PENDING",
    "note": "Created via API"
  }'

3) Обновить платеж
curl -X PUT 'http://localhost:8080/api/payments/123e4567-e89b-12d3-a456-426614174000' \
  -H 'Content-Type: application/json' \
  -d '{
    "inquiryRefId": "123e4567-e89b-12d3-a456-426614174005",
    "amount": 300.00,
    "currency": "RUB",
    "transactionRefId": "123e4567-e89b-12d3-a456-426614174006",
    "status": "COMPLETED",
    "note": "Updated and completed"
  }'

4) Обновить заметку платежа
curl -X PATCH 'http://localhost:8080/api/payments/123e4567-e89b-12d3-a456-426614174000/note' \
  -H 'Content-Type: text/plain' \
  -d 'Платёж подтверждён. Ожидается зачисление.'

5) Удалить платеж
curl -X DELETE 'http://localhost:8080/api/payments/123e4567-e89b-12d3-a456-426614174000'
